# Overview

Using Claude, ChatGPT, Gemini, and Grok to Create 16 Browser Games
This repository documents an experiment exploring how well modern AI models can independently generate complete, playable video games.
Each model was given the same initial prompt to generate an HTML game inspired by the game Vampire Survivors.
The goal was to evaluate each model’s ability to:

* Generate runnable HTML/CSS/JavaScript code

* Maintain internal consistency across files

* Deliver something fun and playable with minimal human intervention

# Models
Four AI models were tested:

* Claude

* ChatGPT

* Gemini

* Grok
